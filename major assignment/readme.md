# 🎵 VibeStream – Music Streaming Mobile App UI/UX

## 📌 Project Description

VibeStream is a modern music streaming mobile application designed to provide new users with a simple, intuitive, and engaging music listening experience. Inspired by popular streaming platforms such as Spotify and Apple Music, the application focuses on clean navigation, attractive visuals, and user-friendly interactions.

The project was created as part of a UI/UX Design Internship assignment using Figma.

---

## 🎯 Project Goal

Design three interactive mobile screens that allow users to:

- Get introduced to the application
- Create an account or log in
- Browse and discover music easily

The objective was to create an interface that is visually appealing while following fundamental UI/UX principles such as consistency, accessibility, hierarchy, and usability.

---

## 🧩 Problem Statement

Many music streaming applications offer numerous features that can overwhelm first-time users. Complex navigation, cluttered interfaces, and inconsistent layouts reduce the overall user experience.

VibeStream aims to solve these problems by providing a clean, minimal, and intuitive interface that enables users to quickly start listening to music with minimal effort.
